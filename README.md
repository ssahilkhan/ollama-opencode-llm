# Ollama OpenCode LLM Project

A web-based interface for interacting with Ollama-powered LLMs locally. Provides a clean, responsive chat UI to communicate with locally hosted language models.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Ollama (Local LLM backend)

## Features

- Local LLM interaction via Ollama
- Clean chat-style interface
- Responsive design
- Real-time message streaming
- Lightweight and fast (no frameworks needed)
